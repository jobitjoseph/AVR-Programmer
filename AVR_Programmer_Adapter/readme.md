# AVR Programmer Adapter
AVR programmer adapter for 8-pin ATtiny microcontrollers for converting:
- 6-pin ICSP connectors,
- 10-pin ICSP connectors,
- 8-pin SOP8/SOIC8 adapters,
- 8-pin SOP8/SOIC8 clips,
- 8-pin DIP8 chips.
